QRCode Almate Shop

QRCode Almate Shop - service for selling stands and cards with a QR code with a built-in NFC chip, which allow you to quickly go to a page on a social network.

Implemented

  • Personal account implemented.
  • Link redirection service created.
  • Admin panel created.
  • Step-by-step activation and card setup implemented.

Technologies / tools

  • PHP, Laravel, MySql, phpMyAdmin, JavaScript, PhpStorm, Git, Bitbucket, CSS, Tailwind CSS.
